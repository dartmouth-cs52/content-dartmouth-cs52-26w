VITE

Vite is a powerful development server and build tool. What it does is take all your dependencies (all the various files and libraries that your project is using), processes them, and then either serves them in dev mode or builds them for production. It does this in a smart way, where it will only bundle up the things you need (with Rollup). It can do any preprocessing that you need, such as converting various code syntax, and supports hot-reloading, so you never have to refresh the page manually again. Most cleverly, it actually analyzes your code and figures out what you are doing and only includes what is necessary.
