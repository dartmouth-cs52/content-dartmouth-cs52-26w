Where in the Browser?
 
* anatomy of a browser * **user interface**:address bar, back/forward button, etc. except the window with page * **browser engine**: sits between the UI and the rendering engine - passes commands * **rendering engine** : displays requested content. parses HTML and CSS, and displays the parsed content on the screen * **Networking**: for network calls such as HTTP requests * **UI backend**: used for drawing basic widgets like combo boxes, main windows, menus * **JavaScript interpreter**. Used to parse and execute JavaScript code. * **Data storage**. Browser local storage like Cookies
